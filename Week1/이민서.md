# Problem_Title

 - 문제 링크
  화씨와 섭씨 온도간의 변환

  1. 섭씨 30℃는 화씨로 ℉인지 ?
  2. 화씨 150℉는 섭씨로 ℃인지 ?

### 풀이 과정  



### 소스 코드

```Language
C1 = 30
F1 = C1*9 / 5 + 32
print('섭씨 30도는 화씨로 %.2f도이다.'%(F1))

F2 = 150
C2 = (F2 - 32) *5 / 9
print('화씨 150도는 섭씨로 %.2f도이다.'%(C2))

```

### 결과 화면
섭씨 30도는 화씨로 86.00도이다.
화씨 150도는 섭씨로 65.56도이다.
