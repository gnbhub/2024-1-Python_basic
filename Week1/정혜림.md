# Problem_Title

 - 문제 링크
1.섭씨 30℃는 화씨로 ℉인지 ?
2.화씨 150℉는 섭씨로 ℃인지 ?
  

### 풀이 과정  



### 소스 코드

```Language
C=37
F=C*(9/5)+32

f=150
c=(f-32)*(5/9)

print(f'섭씨가 {C}일 때 화씨:{F:.2f}\n화씨가 {f}일 때 섭씨:{c:.2f}')

```

### 결과 화면
섭씨가 37일 때 화씨:98.60
화씨가 150일 때 섭씨:65.56
